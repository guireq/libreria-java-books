Java Books - Sistema de Gestión de Libros

Flujo de Autorización OAuth2

Para acceder a la API de libros, necesitas obtener un token OAuth2 usando el Authorization Code Flow.

Paso 1: Autorización

Accede a la siguiente URL para obtener el código de autorización:

http://localhost:9000/oauth2/authorize?response_type=code&client_id=javabooks-client&scope=libros.read%20libros.write&redirect_uri=http://localhost:8080/authorized&code_challenge=PKCE_CHALLENGE&code_challenge_method=S256
Endpoints de la API:
  • POST /api/book - Crear libro (requiere ADMIN y scope libros.write)
  • DELETE /api/book/{id} - Eliminar libro (requiere ADMIN y scope libros.write)
  • GET /api/book/title/{title} - Buscar por título (requiere scope libros.read)
  • GET /api/book/author/{author} - Buscar por autor (requiere scope libros.read)
