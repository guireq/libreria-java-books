¡Autorización Exitosa!

Has sido autorizado correctamente. Ahora puedes usar el código de autorización para obtener tu token de acceso.

Siguiente paso:

Usa el código de autorización que aparece en la URL para intercambiarlo por un token de acceso realizando una petición POST a:

POST http://localhost:9000/oauth2/token
Parámetros requeridos:
  • grant_type=authorization_code
  • client_id=javabooks-client
  • client_secret=secret
  • code=[código de la URL]
  • redirect_uri=http://localhost:8080/authorized
